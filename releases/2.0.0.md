# Cloud.ru Evolution Terraform Provider - major release 2.0.0

## Введение
Мы рады представить обновленный Cloud.ru Evolution Terraform provider версии 2.0.0!

В новой версии произошли существенные изменения. Провайдер теперь поддерживает значительно больше
продуктов и ресурсов облака Cloud.ru

## Обратная совместимость
Новая версия 2.0.0 **не совместима** с версией 1, ввиду значительных изменениях в схемах конфигураций ресурсов и подходе взаимодействия с облаком.

Ресурсы провайдера версии 1.x.x НЕ ПОДДЕРЖИВАЮТСЯ в провайдере версии 2.0.0.

## Имеющиеся проблемы
`Import state` - данная функциональность на данный момент работает не корректно.
Миграцию ресурсов с 1 версии провайдера на данный момент производить нельзя. Не импортируйте ранее созданные ресурсы в состояние (import state)!\
Дождитесь, пожалуйста, исправления в ближайших патч-версиях.

Полная поддержка функции import state для всех ресурсов будет реализована в Q2 2026
Параллельная работа с ui/api не поддерживается. Т.е. если вы управляете ресурсами через терраформ, то создавайте и редактируйте их только через тераформ, иначе стейт ресурса может повредиться.
