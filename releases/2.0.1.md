# Cloud.ru Evolution Terraform Provider - patch 2.0.1

## Изменения

- Перенесли MCM из 1.6 c тем же функционалом
- Добавили заголовки для трейсинга

## Исправления

- Исправили потенциальную возможность, что compute поля могут быть не заполнены, вызывая ошибку провайдера
- Исправили ошибку, когда параметр interface_security_enabled в ресурсе интерфейса не менялся
- Исправили ошибку, когда при удалении стейт мог очиститься даже если сам ресурс не был удален
- Исправили ошибку, когда повторный terraform apply показывал изменения по полям policy, quota, id, uuid для бакета в s3 даже если конфигурация ресурса не менялась.
- Исправили ошибку, когда при неверных параметрах Compute ресурсов пользователь видел только общее сообщение об ошибке без деталей.
