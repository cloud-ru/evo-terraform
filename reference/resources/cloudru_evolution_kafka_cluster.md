## Schema

### Required

- `brokers` (Number) Количество брокеров в кластере.
- `cruise_control_enabled` (Boolean) Укажите `true`, чтобы включить Cruise Control для управления кластером.
- `cruise_control_rebalance_auto_enabled` (Boolean) Укажите `true`, чтобы включить автоматическое перераспределение партиций в кластере.
- `name` (String) Имя кластера.
- `project_id` (String) Идентификатор проекта.
- `specification_id` (String) Идентификатор спецификации кластера.
- `storage_gib` (Number) Размер диска в гигабайтах.
- `subnet_id` (String) Идентификатор подсети.
- `ui_enabled` (Boolean) Включен ли визуальный интерфейс кластера.
- `version_id` (String) Идентификатор версии кластера.

### Optional

- `description` (String) Описание кластера.
- `logging` (Attributes) Параметры интеграции с сервисом Клиентского логирования. (see [below for nested schema](#nestedatt--logging))
- `parameters` (Map of String) Параметры кластера <имя параметра, значение>. Список поддерживаемых параметров и их описание доступны в разделе документации [Параметры кластера](https://cloud.ru/docs/paas-kafka/ug/topics/guides__parameters__available-parameters-list).

### Read-Only

- `created_at` (String) Дата создания кластера.
- `health` (String) Состояние здоровья кластера.
- `id` (String) Идентификатор кластера.
- `in_sync_replicas` (Number) Количество реплик, синхронизированных с лидером.
- `kafka_hosts` (List of String) Адреса кластера.
- `kafka_ui_host` (String) Адрес UI-интерфейса кластера.
- `status` (String) Статус кластера.
- `topic_partitions` (Number) Количество партиций.
